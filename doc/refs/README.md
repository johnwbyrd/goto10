# Reference Materials

## c64_rnd_disassembly.txt

Hand-curated extract of the RND function disassembly from the C64 BASIC ROM ($E097-$E0F6), plus the FMULT constants and supporting routines. Created early in this project from the pagetable.com disassembly. Some comments are outdated — the claim that FADD "does nothing" is wrong.

## c64disasm_en_davison.txt

Lee Davison's complete commented disassembly of the C64 BASIC and Kernal ROMs. Used for tracing instruction-level behavior during debugging.

## rom_src/

Original Microsoft BASIC 6502 source code. Key files for the RND analysis:

- **declare** — Zero-page variable declarations. Defines addresses for FAC ($61-$66), FACOV ($70), BITS ($68), RNDX ($8B-$8F), RESHO-RESLO ($26-$29), and ARG ($69-$6D).
- **code17** — FADD/FSUB entry points, NORMAL (normalize) routine.
- **code18** — SHIFTR/SHFTR2/MULSHF (right-shift routines), FADD2 (addition with carry), NORM2 (bit-level normalize), SQUEEZ/RNDSHF (overflow handling).
- **code19** — FMULT (multiply), MLTPLY/MLTPL1 (shift-and-add inner loop), CONUPK (unpack constant into ARG), MULDIV (combine exponents).
- **code20** — MOVFR (copy result to FAC), MOVFM (unpack memory to FAC — clears FACOV), MOVMF (round and pack FAC to memory).
- **code25** — RND function entry point.

## vice_1m_seeds.txt

One million consecutive RND(1) seed values captured from VICE 3.9 running the actual C64 ROM. Generated by `c64/rnd_spy.s` (pure 6502 assembly calling the ROM's RND routine directly, no BASIC interpreter involvement). Format: `BITS FACOV EXP M1 M2 M3 M4` in hex, one line per call. This is the ground truth for validating the C++ simulation.
